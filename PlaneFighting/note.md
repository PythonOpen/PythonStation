# v2
- 小蜜蜂会动，从上往下慢慢的飞
- 能控制小蜜蜂左右移动
- 入场算法
    - y轴要求是负数，这样会形成入场的效果，y = 0 - 蜜蜂的高度
    - x轴要求是有一定的富余，即要求蜜蜂等移动物品不能隐藏起来
        - 基本上x轴的值应该是50起，最右边的计算方式world的宽度-蜜蜂的宽度
    - 移动速度问题
        - x, y轴的速度
        - 对于绝大多数物体，则只考虑y值
        - 但是，密封和英雄机是特例
        - 密封从上往下移动的同时具有横向运动
        - 英雄机的移动由上下左右键盘控制
        - 速度应该是一个tuple=(x,y)
    - 方向问题
        - 具体移动方向由x, y控制
        - 值只能是-1， 0， 1 三个就好
        - 应该是一个tuple
        - 例如(-1, 0)标识水平向左移动
        - (0,1)表明向下垂直运动
